# 💕 Exam Prep Quiz App 💕

Interactive and Beautiful Quiz Application for Exam Preparation

## 📚 Features

- **450 MCQs** across 3 chapters (12, 13, 14)
- **5 marks per question** (Total: 750 marks)
- **60-second timer** per question
- **Instant feedback** with explanations
- **Comprehensive grading system** (A+ to F)
- **Pass/Fail status** (60% passing)
- **Beautiful UI** with animations
- **Responsive design** for all devices

## 🎯 Chapters

1. **Chapter 12**: AI Agent Factory Paradigm (150 MCQs)
2. **Chapter 13**: Markdown - Writing Instructions (150 MCQs)
3. **Chapter 14**: Claude Code and Cowork (150 MCQs)

## 🏆 Grading System

- **A+**: 90%+ (675+ marks)
- **A**: 80-89% (600-674 marks)
- **B**: 70-79% (525-599 marks)
- **C**: 60-69% (450-524 marks)
- **D**: 50-59% (375-449 marks)
- **F**: Below 50% (Below 375 marks)

## 🚀 How to Use

1. Open `index.html` in your browser
2. Select a chapter to start
3. Answer questions within 60 seconds each
4. Get instant feedback and explanations
5. View detailed results at the end
